# Back Home

**Back Home** is a Unity game project developed as a final game design/development project.  
The game focuses on guiding the player through the experience of trying to return home while interacting with the environment, following the game’s rules, and overcoming its mechanics-based challenges.

## 🎮 About the Game

In **Back Home**, the player must navigate the game world, understand the rules of the environment, and use movement and interaction mechanics to progress. The project includes gameplay systems, visual assets, and documented game rules/mechanics.

## 🛠️ Built With

- Unity
- C#
- ShaderLab / HLSL
